# Naive Bayes Classification

This project demonstrates the implementation of a Naive Bayes classifier using the `covid_toy.csv` dataset to predict whether a person has COVID-19 based on demographic and health-related features.

## Dataset

* **File:** `covid_toy.csv`
* **Task:** Binary Classification

## Model

* **Algorithm:** Naive Bayes
* **Library:** Scikit-learn

## Performance

* **Accuracy:** 0.60 (60%)

## Features Used

* Age
* Gender
* Fever
* Cough
* City

## Files

* `naive_bayes.ipynb` – Model training notebook
* `NaiveBayesModel.pkl` – Saved trained model
