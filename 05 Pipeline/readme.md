# Social Network Ads Prediction using Pipeline

This project predicts whether a user will purchase a product based on Age and Estimated Salary using a Scikit-Learn Pipeline.

## Technologies Used
- Pandas
- NumPy
- Scikit-Learn

## Pipeline Steps
1. StandardScaler
2. PCA
3. Random Forest Classifier

## Dataset
Social_Network_Ads.csv

## Accuracy
**88.75%**

## Features Used
- Age
- Estimated Salary

## Target
- Purchased (0 = No, 1 = Yes)

## Concepts Learned
- Data Preprocessing
- Feature Scaling
- PCA
- Pipeline
- Random Forest Classification
- Model Evaluation
