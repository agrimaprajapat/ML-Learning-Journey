# Random Forest

This folder contains implementations of Random Forest for both classification and regression tasks.

## Datasets
- **Classification:** `covid_toy.csv`
- **Regression:** `insurance.csv`

## Models
### Random Forest Classifier
- Dataset: `covid_toy.csv`
- Accuracy: **0.60**

### Random Forest Regressor
- Dataset: `insurance.csv`
- R² Score: **0.86**

Practice implementing Random Forest algorithms for classification and regression problems using Scikit-learn.
