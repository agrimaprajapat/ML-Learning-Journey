# Market Basket Analysis using Apriori

A simple implementation of the Apriori algorithm for Market Basket Analysis to discover products that are frequently purchased together.

## Tools Used
- Python
- Pandas
- Apyori

## Dataset
- 7501 transactions
- 20 items per transaction

## Steps
1. Data preprocessing
2. Transaction creation
3. Apriori algorithm training
4. Association rule generation

## Output
Generated association rules with support, confidence, and lift metrics.
