# 🛍️ Mall Customer Segmentation using K-Means

Customer segmentation project using K-Means Clustering on the Mall Customers dataset.

## Dataset
- mall.csv

## Features Used
- Annual Income (k$)
- Spending Score (1-100)

## Concepts Covered
- Unsupervised Learning
- K-Means Clustering
- Elbow Method
- Customer Segmentation
- Cluster Visualization

## Results
- Optimal Clusters: 4
- Customers grouped based on spending behavior and annual income.

## Libraries
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
