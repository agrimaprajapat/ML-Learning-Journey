# Iris Flower Classification using SVM

This project implements a Support Vector Machine (SVM) classifier on the Iris dataset using Scikit-learn.

## Dataset
- Built-in Iris dataset from Scikit-learn
- 150 samples
- 4 features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- 3 target classes:
  - Setosa
  - Versicolor
  - Virginica

## Algorithm
- Support Vector Machine (SVM)
- RBF Kernel

## Libraries Used
- NumPy
- Pandas
- Scikit-learn

## Results
- Model Accuracy: **100% (1.0)**

## Steps Performed
1. Loaded the Iris dataset.
2. Split the data into training and testing sets.
3. Trained an SVM classifier with an RBF kernel.
4. Made predictions on the test set.
5. Evaluated the model using accuracy score.

## Outcome
The SVM model successfully classified Iris flower species with high accuracy on the test dataset.
